# Overview

Profitist — персональный ИИ-ассистент в виде Telegram-бота.

## Ключевая идея

Не чат-бот «запрос → ответ», а **проактивный агент**: помнит контекст о пользователе через сессии, принимает задачи на исследование, выполняет их асинхронно и сам присылает результаты.

## Поведение

- Пользователь пишет «запомни, что я строю стартап» → агент сохраняет факт
- Пользователь пишет «исследуй тренды fintech и пришли завтра» → агент планирует задачу, выполняет ресёрч, отправляет сам
- Агент раз в день проверяет контекст и может написать первым, если есть что-то ценное

## Структура проекта

```
src/profitist/
├── config.py          — настройки (pydantic-settings)
├── main.py            — точка входа
├── db/                — SQLAlchemy модели + engine
├── memory/            — CRUD поверх БД (единственная точка доступа к данным)
├── agent/             — Claude agentic loop, роутинг моделей, инструменты
├── bot/               — aiogram handlers + middleware
├── scheduler/         — APScheduler: фоновые и проактивные задачи
└── search/            — обёртка над Tavily
```

## Стек

| Слой | Технология |
|---|---|
| Telegram | aiogram 3.x |
| LLM | Anthropic SDK (claude-haiku-4-5 / claude-sonnet-4-6) |
| БД | PostgreSQL + SQLAlchemy 2.x async + Alembic |
| Планировщик | APScheduler 3 AsyncIOScheduler |
| Web search | tavily-python |
| Конфиг | pydantic-settings |
